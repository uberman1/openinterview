HOW TO DOWNLOAD FROM REPLIT:
-----------------------------

Option 1: Direct Download (Recommended)
1. Click on "openinterview-application.tar.gz" in the file tree
2. Click the "..." menu (three dots) in the file viewer
3. Select "Download"
4. Save to your computer

Option 2: Multiple Files
1. Download openinterview-application.tar.gz
2. Download DOWNLOAD_PACKAGE_README.md
3. Download PACKAGE_INVENTORY.txt (optional)

Option 3: Git Clone
1. If this is a Git repository, clone it:
   git clone <repository-url>

AFTER DOWNLOAD:
---------------
1. Extract: tar -xzf openinterview-application.tar.gz
2. Install: npm install
3. Configure: Create .env file with required variables
4. Run: npm run dev
