# Playwright Config Patch (CI-Stable Painting)

This bundle hardens Playwright for CI so Chromium reliably paints pages that went blank in headless runs.

## Contents
- `playwright.config.js`
- `tests/editor.spec.ts`
- `tests/template-workflow.spec.ts`
- `package.json` (scripts incl. headed)
- `README_PATCH.md` (this file)

## Use
1) Unzip at repo root.
2) Adjust routes if needed.
3) Install & run:
   npm install
   npx playwright install --with-deps
   npm run serve
   npm run test:e2e
