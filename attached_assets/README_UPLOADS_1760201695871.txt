Install & Run (Replit)
1) unzip at repo root:
   unzip uploads_pack_v0.1.0.zip -d .

2) install deps:
   pip install -r uploads_pack/requirements.txt
   python -m playwright install --with-deps chromium

3) optional: point to hosted preview
   # export OI_BASE_URL="https://<replit-preview-domain>"

4) run tests:
   python uploads_pack/run.py
