# Availability — Vertical Weekly Layout Patch
Replaces the Weekly Hours grid with a **vertical day list** (Calendly-style) while preserving all existing behavior.

## Apply in Replit
```bash
unzip availability_vertical_weekly_patch.zip -d /tmp/oi_avv && cp -R /tmp/oi_avv/* .
node scripts/apply_availability_vertical_patch.mjs
# restart server (e.g., npm run dev)
node scripts/selftest_availability_dom_vertical.mjs
```

## Notes
- Uses the same APIs (`/api/availability`, `/api/slots`) from the previous module.
- Day rows contain:
  - **+** Add (day-level) when empty
  - For each block: time inputs + **×** remove, **+** add another block, **⧉** copy block set to all weekdays
- Timezone selector moved into the Weekly card (like reference).

## Guardrails
- This patch updates `expected.sha256.json` (if present) with the new file checksums so guardrails stay enforced.

## Rollback
```bash
cp -f public/availability_prev.html public/availability.html
cp -f public/js/availability_prev.js public/js/availability.js
```
