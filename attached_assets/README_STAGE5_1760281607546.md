Turn your harness into a Pilot/UAT facilitator:

## Quick Start
unzip Stage5_Pilot_UAT_v0_5_0.zip -d .
python scripts/apply_stage5_gate_patch.py
export HEALTH_URL="http://127.0.0.1:8000/health"; export EXPECT_LIVE=1
PYTHONPATH=. python stage5/run_stage5.py
python scripts/update_test2_index_stage5.py

Open `/uat_hub.html` to record feedback. To export combined feedback placed at `qa/stage5/feedback_local.json` into individual files:
python stage5/export_feedback.py
