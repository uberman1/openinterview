# OpenInterview – Self‑Deploying Bundle

This bundle updates the profile editor:

- **public/profile_edit_enhanced.html**
  - Adds an **Edit / Publish** banner.
  - Adds **Profile Name** field (writes to `display.name`).
  - Wires Edit (stay on editor with `?id=`) and Publish (calls `store.publishProfile(id)` and shows the share link).
- **public/js/profile_edit.bind.js**
  - Rules updates:
    - Removes **Scheduling Window** + **Increments**.
    - **Minimum Notice** supports **Immediate (0h)** and **Custom** hours.
    - **Daily Cap**: blank = **unlimited** + explainer text.
    - **Durations**: adds **45 min** and **120 min** (kept in model).
    - **Buffers**: adds explainer text.
  - Keeps your nested `display` model and `store.*` API.

## One‑shot deploy on Replit

1) Upload this ZIP to your Replit project and open the Shell.
2) Run:
```bash
unzip -o openinterview_self_deploy.zip && bash deploy.sh
```
3) Start your app. Visit:
```
/public/profile_edit_enhanced.html?id=<yourProfileId>
```
Verify:
- Top banner buttons behave as expected.
- Name field persists and appears under “My interviews” on Home.
- Rules UI reflects the requested changes and save logic.
