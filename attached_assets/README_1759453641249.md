# Foundations Pack (Modules 0–2)
This bundle is optimized for Replit limits: one runner, one utils folder, flat tests directory. Each module has guardrails, automated journeys, and a human QA checklist.

Run once per Replit:
  npm i
  npx playwright install --with-deps
  cp .env.example .env  # set BASE_URL

Run a single module (example: Module 0):
  MODULE=module0 npm run journey

Run all modules (0→2):
  npm run journey:all

Logs:
  logs/journey-log-<MODULE>-YYYYMMDD-HHMMSS.ndjson
Screenshots:
  artifacts/screenshots/<MODULE>/*.png

==================================================
Module 0 — Scaffolding & Demo Mode
==================================================
Goal: All routes render in Demo Mode so screens are visible without auth or data.

Guardrails:
- /demo.html loads and lists links to primary routes.
- Protected routes render with ?demo=1.
- No fatal page errors during navigation.

Human QA:
- Open /demo.html, click Dashboard, Profile, Settings, Jobs.
- Verify a visible UI renders for each.
- Confirm console shows no repeating errors.

Automated:
  MODULE=module0 npm run journey

==================================================
Module 1 — Navigation Shell & Page Stubs
==================================================
Goal: App has a global header/sidebar and each page shows placeholder content.

Guardrails:
- Header/sidebar visible on protected routes.
- Clicks change routes without full reload.
- No console errors on page switch.

Human QA:
- Manually click all nav links several times.
- Verify active state updates and content area changes.

Automated:
  MODULE=module1 npm run journey

==================================================
Module 2 — Auth Simulation & Redirects
==================================================
Goal: Simulated login redirects to Dashboard, logout returns to Login.

Guardrails:
- Visiting /dashboard when not “logged in” redirects to /login.
- Fake login forwards to /dashboard and persists for that session.
- Logout clears state and returns to /login.

Human QA:
- Attempt /dashboard unauthenticated → redirected to /login.
- Perform fake login → land at /dashboard.
- Logout → back to /login.

Automated:
  MODULE=module2 npm run journey
