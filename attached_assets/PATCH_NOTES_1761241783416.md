This patch:
1) Removes **Edit** / **Publish** from the *edit* page (`profile_edit_enhanced.html`).
2) Adds **Save & Return** on the edit page (saves then returns to template screen).
3) Adds **Edit**, **Save & Publish**, and **Share** to the *template* page (`profile_v4_1_package/public/index.html`).

## Install
1. Unzip into your Replit project root. It will overwrite:
   - `profile_edit_enhanced.html`
   - `profile_v4_1_package/public/index.html`
2. Reload your app.

## Behavior
- **Edit (template page):** navigates to `profile_edit_enhanced.html?id=...`.
- **Save & Publish (template page):** publishes profile **in place** and updates status label to `LIVE`.
- **Share (template page):** opens a modal with a shareable URL, copy, native share (when supported), and email link.
- **Save & Return (edit page):** triggers save via a generic `profile:save-request` event and returns to the template page.

## Requirements
- `/js/data-store.js` should expose `getProfile(id)` and `publishProfile(id)`.
- Tailwind is served locally via `/css/tailwind.css`.
