# Availability Patch — Update Grid Step Description & Remove Preview Tab

This patch:
- Changes the **Start time increments** description to:
  > Your interview slot intervals. If the grid step is 30 min, possible start times are 9:00, 9:30, 10:00, 10:30…
- Removes the **Preview** tab and its section from `availability.html`.
- Removes any `preview` handling in `public/js/availability.js` (if present).
- Updates `expected.sha256.json` checksums (if the guardrail file exists).

## Apply (Replit)
```bash
unzip availability_remove_preview_and_update_gridstep_patch.zip -d /tmp/oi_upd && cp -R /tmp/oi_upd/* .
node scripts/apply_remove_preview_and_update_gridstep.mjs
# restart server (e.g., npm run dev)
npm i supertest cheerio
node scripts/selftest_remove_preview_and_update_gridstep.mjs
```

cp -f public/availability_prev.html public/availability.html
