# Top Menu Update v3 (Availability-first)

**What changes**
- Moves the unified menu **above** the page header/title/buttons on `availability.html` (so it sits at the very top line).
- Continues to remove “New Profile” and “New Interview” buttons.
- Keeps the same Profiles-style menu across: Home, Availability, Profiles, Uploads, Subscription, Password, Log Out.
- Uses response-time injection (guardrail-safe).

## Install (Replit)
```bash
unzip top_menu_update_v3.zip -d /tmp/tmv3 && cp -R /tmp/tmv3/* .
node scripts/apply_top_menu_update_v3.mjs
# restart server (e.g., npm run dev)
npm i supertest cheerio
node scripts/selftest_top_menu_update_v3.mjs
```

**Note:** DOM placement (above header content) is handled client-side only on `availability.html`.
