# Home Page Design Update — Patch
- Replaces Home with your **exact HTML** (design-focused).
- Removes old top buttons (Share Profile / Manage Profiles) by omission and defensively at runtime.
- Wires **Home / Explore / My Profile** links and **New Interview** button to existing routes without changing the HTML (binder injection).

## Apply (Replit)
```bash
unzip home_page_design_update_patch.zip -d /tmp/oi_home && cp -R /tmp/oi_home/* .
node scripts/apply_home_page_design_update.mjs
# restart server (e.g., npm run dev)
npm i supertest cheerio
node scripts/selftest_home_page.mjs
```

## Binder routing
- **Home** → first existing among `/home.html`, `/index.html`, `/`
- **Explore** → first existing among `/profiles.html`, `/profiles`, `/explore.html`
- **My Profile** → first existing among `/profile.html`, `/profile_home.html`, `/me.html`
- **New Interview** → first existing among `/interviews/new`, `/profile_edit.html?new=1`, `/profile_edit.html`

No new endpoints added; uses your current app.

## Rollback
```bash
cp -f public/home_prev.html public/home.html
```
