# OpenInterview – Upcoming Interviews: Contact Column Patch (Replit Self‑Deploy)

This package safely patches your **home.html** (and any JS that renders the table) to:
1) Rename the column header **Actions → Contact**
2) Replace **"View Details"** links with a **static recruiter email** (plain text, no hyperlink)

**Run on Replit:**
```bash
bash run.sh
```
For custom repo path:
```bash
python3 scripts/patch_upcoming_contact.py --repo /home/runner/your-project
python3 scripts/self_test.py --repo /home/runner/your-project
```
