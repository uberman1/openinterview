# OpenInterview – Playwright E2E Suite (Replit-ready)

Run an automated Playwright battery validating Phase 1–3 on Replit.

## Run steps
1) Install:
   npm install --save-dev @playwright/test
   npx playwright install
2) Serve the app (new tab):
   npx serve -l 3000 .
3) Run tests:
   npm run test:e2e
