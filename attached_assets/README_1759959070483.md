# OI-UI-Scroll-Hero-v1 — Binder (Guardrail-Safe)

Adds **hero shrink to 75% on scroll** and **below-resume scrolling** without editing protected HTML.

## Deploy
```bash
unzip oi-ui-scroll-hero-v1-binder.zip -d oi-binder
cd oi-binder
bash deploy.sh /absolute/path/to/repo
```

Then wire one of the options below.

### Option A — Express injection (recommended)
In your `server.js`:
```js
const express = require('express');
const path = require('path');
const app = express();

app.use('/js', express.static(path.join(__dirname, 'public', 'js')));
require('./server/examples/servePublicProfile.inject.example.js')(app);

app.listen(process.env.PORT || 3000);
```
Visit `/public_profile.html` to verify.

### Option B — Global bundle
If a non-protected bundle is already on the page, import:
```js
import '/js/public_profile.hero-shrink.bind.js';
```

## Rollback
Remove the script tag / import. Nothing is modified on disk.

## What it does
- Injects scoped CSS at runtime.
- Adds IDs/classes to the hero.
- Wraps content **after** Resume into a scrollable container (desktop).
- Smoothly shrinks hero to **75%** over ~240px scroll; restores at top.
- Honors **prefers-reduced-motion**.
