# Uploads Page Replacement — Replit Patch
**No HTML changes.** Exact UI provided is installed as-is and wired at response-time.

## Apply
```bash
unzip replit_uploads_replacement_patch.zip -d /tmp/oi_up && cp -R /tmp/oi_up/* .
node scripts/apply_uploads_patch.mjs
npm i supertest cheerio
# restart server (e.g., npm run dev)
node scripts/selftest_uploads_dom.mjs
```

## Integration (uses existing endpoints only)
- List (resumes/general): `GET /api/uploads?type=resume|general`
- Upload (multipart): `POST /api/uploads`  fields: `file`, `type`
- View: `GET /api/uploads/:id/view`
- Download: `GET /api/uploads/:id/download`
- Delete: `DELETE /api/uploads/:id`
- Client size limits: Resumes ≤ **10MB**, General ≤ **25MB** (docs & slide decks only)
