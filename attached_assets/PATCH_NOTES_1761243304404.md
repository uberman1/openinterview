# Patch: Match "Save & Publish" button style to Edit/Share

This patch makes the **Save & Publish** button use the same ghost style as **Edit** and **Share**.

## Files affected
- `profile_v4_1_package/public/index.html`

## What changes
- Replaces `bg-primary text-white text-sm font-bold` with
  `border border-subtle-light dark:border-subtle-dark bg-background-light dark:bg-subtle-dark text-sm font-semibold`
- Adjusts padding from `px-5` to `px-4` to match siblings.

## How to apply (git)
```bash
# from your project root
git apply 0001-match-publish-button-style.patch
# or: git am < 0001-match-publish-button-style.patch
```

## Manual fallback (no git)
Open `profile_v4_1_package/public/index.html` and replace:
```html
<button id="btnTplPublish" data-testid="button-template-publish" class="h-10 px-5 rounded-lg bg-primary text-white text-sm font-bold">Save &amp; Publish</button>
```
with:
```html
<button id="btnTplPublish" data-testid="button-template-publish" class="h-10 px-4 rounded-lg border border-subtle-light dark:border-subtle-dark bg-background-light dark:bg-subtle-dark text-sm font-semibold">
  Save &amp; Publish
</button>
```
