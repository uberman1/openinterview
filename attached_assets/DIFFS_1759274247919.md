# Diffs to update static path (../dist/public)

Update your server static path to point at the built client: `../dist/public`.

---
## /server/vite.ts  (or equivalent static-serve helper)

**Before (example):**
```ts
const clientDir = path.resolve(__dirname, './public')
if (!fs.existsSync(clientDir)) {
  throw new Error(`Could not find the build directory: ${clientDir}, make sure to build the client first`)
}
app.use(express.static(clientDir))
```

**After:**
```ts
const clientDir = path.resolve(__dirname, '../dist/public')
if (!fs.existsSync(clientDir)) {
  throw new Error(`Could not find the build directory: ${clientDir}, make sure to build the client first`)
}
app.use(express.static(clientDir))
```

---
## /server/index.ts  (only if it references the old public path)

Search for `./public` or `server/public` and replace with `../dist/public`.

**Example:**
```ts
// Before
const clientDir = path.resolve(__dirname, './public')

// After
const clientDir = path.resolve(__dirname, '../dist/public')
```
