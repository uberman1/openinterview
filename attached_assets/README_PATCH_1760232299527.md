
# Patch: Wire Bundle A v0.2.0 into Release Gate + Update test2.html

## Steps
1) Unzip:
```bash
unzip -o bundle_a_gate_patch_v0_2_0.zip
```
2) Patch the release gate:
```bash
python scripts/apply_bundle_a_gate_patch.py
```
3) Ensure Bundle A deps are installed:
```bash
pip install -r bundle_a/requirements.txt
```
4) Start backend in a separate terminal:
```bash
bash scripts/serve_api.sh
```
5) Run Bundle A and save artifacts:
```bash
bash bundle_a/run_and_save.sh
```
6) Update `test2.html`:
```bash
python scripts/update_test2_index.py
```
7) (Optional) Run the full release gate:
```bash
PYTHONPATH=. python release_gate/run_all.py
```
CI example: `ci/snippets/bundle_a_gate.yml`
