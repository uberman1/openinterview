Removes the **duplicate Attachments section** from your home.html file.

## How to use
1. Upload this folder into your Replit project root.
2. Run in shell:
   ```bash
   npm install
   npm run apply
   ```
3. A backup copy (home.html.<timestamp>.bak) will be created automatically.
4. The cleaned version replaces your current home.html.
