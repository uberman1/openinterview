# Deploy Steps (Replit)

1) Upload and unzip this bundle into your Replit project root.
2) Ensure this file exists after unzip:
   - `public/profile_edit_enhanced.html`
   - `public/js/profile_edit.bind.js`
3) Apply diffs (choose one):
   A) **Manual edits** to the referenced files using the changes in `patches/*.diff`,
   B) Or replace the files if they are small wrappers.
4) Ensure server route:
   - `GET /profile_edit_enhanced.html` serves the new page.
   - `GET /profile_edit.html` redirects to `/profile/:id/template?templateId=default`.
5) Start the app and test:
   - Create New → lands on template page with dummy data.
   - Click Edit on template → opens enhanced editor.
   - Upload a resume → profile auto-populates.
   - Set availability → Save persists.
