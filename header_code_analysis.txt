========================================
SUMMARY OF HEADER PATTERNS
========================================

COMMON ELEMENTS ACROSS PAGES:
1. Logo: SVG icon (typically h-6 w-6 or size-6) + brand text
2. Navigation: Hidden on mobile (hidden md:flex), gap-6 or gap-8
3. Avatar: Typically w-10 h-10 rounded-full with background-image
4. Spacing: Most use px-10 py-4 or px-6 py-4, with container mx-auto
5. Border: border-b with opacity variations (border-primary/10, border-gray-200, etc.)

VARIATIONS:
- home.html: Has "New Interview" button + avatar
- uploads.html: Different logo SVG, has border on avatar
- subscription.html: No logo SVG, solid color avatar placeholder
- password.html: Has "Sign In" button instead of avatar
- profiles.html: Has "New Profile" button + avatar, 5 nav links
- availability.html: Page header only, no top nav/logo/avatar

DARK MODE:
All pages support dark mode with appropriate dark: prefixes for colors and borders.
